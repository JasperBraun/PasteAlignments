◆ automatic_formatting

bool arg_parse_convert::HelpStringFormat::automatic_formatting {false}

Determines whether settings width, parameter_indentation, and description_indentation are used during help-string generations.

If set to false, the help-string does not enforce any limits on the line width of a help-string and does not automatically indent.

◆ description_indentation

int arg_parse_convert::HelpStringFormat::description_indentation

Determines indentation for the lines in a parameter's help-string which contains the parameter's description.

Only applies if automatic_formatting is set to true. Must be less than width for the help-string to be printed.

◆ parameter_indentation

int arg_parse_convert::HelpStringFormat::parameter_indentation

Determines indentation for the line in a parameter's help-string which contains the parameter's name and default argument placeholder, if any.

Only applies if automatic_formatting is set to true. Must be less than width for the help-string to be printed.

◆ width

int arg_parse_convert::HelpStringFormat::width

Limits the width of the entire help-string.

Only applies if automatic_formatting is set to true. Must be larger than parameter_indentation and description_indentation for the help-string to be printed.
