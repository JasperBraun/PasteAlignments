ParameterConfiguration Class Reference
ArgParseConvert-Reference

Description

Stores various properties of a parameter relevant to a containing ParameterMap object.

Instantiated via Parameter factory methods.

Invariant
Maximum number of arguments is either 0 or greater than or equal to minimum number of arguments.
Object has at least one name.
